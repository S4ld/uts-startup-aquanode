## Deskripsi Proyek
AquaNode Kepri adalah prototipe aplikasi web *Tech-Startup* berupa *dashboard monitoring* kualitas air laut berbasis Internet of Things (IoT). Aplikasi ini dirancang untuk memberikan solusi atas masalah lokal di Kepulauan Riau, khususnya membantu para nelayan dan pembudidaya ikan kelong dalam memantau suhu air, kadar pH, dan Oksigen Terlarut (DO) guna mencegah kematian ikan massal.

Proyek ini dibangun menggunakan *framework* Laravel (PHP) dengan mengimplementasikan pola arsitektur MVC (*Model-View-Controller*). Pengembangan berfokus pada manajemen **Routing** dan **Controller** untuk mengolah dan mendistribusikan data sensor simulasi ke tampilan antarmuka pengguna.

## Fitur Utama
* **Live Dashboard:** Visualisasi data metrik kelautan (Suhu, pH, DO) secara *real-time*.
* **Status Alert:** Indikator peringatan dini (*early warning system*) apabila parameter air berada pada kondisi kritis.
* **Modern UI/UX:** Antarmuka responsif bergaya *dark mode* dengan efek *glassmorphism* menggunakan Bootstrap 5.
